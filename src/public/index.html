Task Manager API guide.


URL Requst Type Description
'/' GET Index page
'/users' POST To insert new user data.
Required : Json data with 'name', 'email' and 'password'.
Returns : Bearer authentication token and saved user data as json response.
'/users/login' POST To login a user.
Required : Json data with 'email' and valid password.
Returns : Bearer authentication token and saved user data as json response.
'/users/me/avatar' POST To upload Avatar for user.
Required : Form-data value 'avatar' with .jpeg/.jpg file attached to it. And an Authorization bearer token of logged in user.
Returns : - NA -
'/users/logout' POST To logout current session of user.
Required : An Authorization bearer token of logged in user.
Returns : - NA -
'/users/logoutAll' POST To logout all the sessions of user.
Required : An Authorization bearer token of logged in user.
Returns : - NA -
'/tasks' POST To create task for a user.
Required : An Authorization bearer token of logged in user. A json body with 'description' and 'completed'(optional) properties.
Returns : Json response of saved task.
'/users/me' GET To read the user profile.
Required : An Authorization bearer token of logged in user.
Returns : Json data with user profile.
'/tasks' GET To read tasks of the user.
Required : An Authorization bearer token of logged in user. Prams like 'completed', 'limit', 'skip', 'sortBy' for Searching, Sorting and Pagination.
Returns : Json data with searched tasks.
'/tasks/{task-id}' GET To read task with its ID.
Required : An Authorization bearer token of logged in user.
Returns : Json data with searched task.
'/users/me' PATCH To update data for user.
Required : An Authorization bearer token of logged in user. A Json containing valid updates.
Returns : Json data with updated user.
'/tasks/{task-id}' PATCH To update task with its ID.
Required : An Authorization bearer token of logged in user. A Json containing valid updates.
Returns : Json data with updated task.
'/users/me' DELETE To delete a user profile.
Required : An Authorization bearer token of logged in user.
Returns : Json data of deleted user.
'/tasks/{task-id}' DELETE To delete a task by ID.
Required : An Authorization bearer token of logged in user.
Returns : Json data of deleted task.
'/users/me/avatar' DELETE To delete a user's avatar.
Required : An Authorization bearer token of logged in user.
Returns : - NA -
